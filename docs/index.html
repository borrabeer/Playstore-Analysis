About

Playstore-Analysis is a project for Problem Solving Information Technology (PSIT) subject. This project is base on Google Playstore. We have analyze data into 4 topic, Rating, Review, Size, and Install.

The topic Rating, Review, Size is about Average of all application in each categories. But the last topic Install is about the most downloads application in Google Playstore that around 1,000,000,000+ install.

Rating

Rating is about average rating of all application in each categories that have on Google Playstore. We have analyze it by gathering data from CSV file and find the average of rating and make it into a chart for easy to understand and readable.

Review

Review is about average review of all application in each categories that have on Google Playstore. Base on how many times that each application have review.

Size

Size is about average of size of all application in each categories that have on Google Playstore. So you will see that the highest average size is in GAME category.

Install

Install is about the application that have the most installs on Google Playstore. So the data that we use have install count into approximate number. The most install application is around 1,000,000,000+ installs.
